# E-Commerce Sales Analysis Using SQL

## Project Overview
This project demonstrates SQL skills using a simulated e-commerce database built in MySQL Workbench. The objective was to design a relational database, populate it with sample data, and perform business analysis using SQL queries.

---

## Database Schema

The project consists of four tables:

### Customers
Stores customer information.

### Products
Stores product details and pricing.

### Orders
Stores customer order information.

### Order Items
Stores product-level details for each order.

## Business Questions Answered

### 1. Which customers generated the highest revenue?

Used GROUP BY and SUM() to calculate total spending per customer.

### 2. Which customers spent more than ₹50,000?

Used HAVING clause to filter customer revenue.

### 3. What is the highest-priced product?

Used a Subquery with MAX().

### 4. How are customers connected to orders?

Used INNER JOIN and LEFT JOIN to analyze relationships between tables.
